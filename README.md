AI ELISA Data Analysis Platform

Project Overview
This is an AI-assisted data analysis platform for ELISA experiments, built with Streamlit and Python.

The tool allows users to analyze experimental data without manual preprocessing, supporting flexible data formats and automated workflows.

Key Features
- Smart column mapping (no need to clean Excel manually)
- Standard curve fitting (Linear & 4PL)
- Multi-group comparison analysis
- Statistical significance testing (t-test)
- Automatic LOD detection
- Interactive visualization

AI Contribution
- Used GPT/Claude for code generation and iteration
- Designed modular workflow through prompt-based development
- Leveraged AI for debugging and optimization

Tech Stack
- Python
- Streamlit
- Pandas / NumPy
- SciPy / Scikit-learn
- Matplotlib

pip install -r requirements.txt
